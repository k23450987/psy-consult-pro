# psy-consult-pro
### **一个心理咨询网站项目**

1.**项目实现的功能**

  * 管理员角色的功能
  
    * 普通用户管理功能（CRUD）
    * 普通用户提出的心理咨询问题的管理（CRUD）
    * 网站内心理测试问题的管理（CRUD）
    * 网站内心理文章、新闻的管理（CRUD、需要用到富文本编辑器）
    * 管理员可以回答普通用户提出的问题
    
  * 用户角色的功能
  
    *  浏览跟心理健康有关的新闻或者文章 
    *  进行心理测试
    *  进行心理咨询、留言

2.**项目中用到的技术**

  * Spring Boot
  * Mybatis
  * Lombok
  * Thymeleaf
